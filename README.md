# The Intersections That Hold a City

*A Network Analysis of Urban Road Resilience*

Graph-theoretic analysis of Pittsburgh's driveable street network: degree distribution, betweenness centrality, small-world properties, and fragmentation under random closure versus targeted removal of critical intersections. Built with OSMnx, NetworkX, Plotly, and Quarto.
